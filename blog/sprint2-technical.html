CSS Positioning

Positioning is super important to understand so lets try to simplify the rules.

Static

The default position for any element is postion: static. When an element has no defined or inherited position property it will be displayed on the page in the order that it was marked up in HTML.

See the Pen Static Positioning by Jared (@logansruin) on CodePen.

Inline VS Block

Inline:

Inline elements are laid out horizontally in the normal flow. Examples of inline elements are:

  • span
  • img
  • a
  • strong

Block:

Block elements are laid out horizontally in the normal flow. Examples of block elements are:

  • div
  • headings H1 to H6
  • p
  • blockquote
