CSS Positioning

Positioning elements on a webpage can be a really challenging concept to grasp, but it is super important if you want to be able to have elements that appear outside of the normal flow that is dictated by HTML.

{ position: static }

When you create a website you will likely have an HTML file, with all of the content for your page, that links to a CSS file which provides the style. HTML has baked in styling to all elements including the default position property. As you have likely guessed the default in HTML is [position: static].

See the Pen Static Positioning by Jared (@logansruin) on CodePen.

Inline vs Block

{ display: inline }

Inline elements are laid out horizontally in the normal flow. Examples of inline elements are:

  • span
  • img
  • a
  • strong

{ display: block }

Block elements are laid out horizontally in the normal flow. Examples of block elements are:

  • div
  • headings H1 to H6
  • p
  • blockquote
